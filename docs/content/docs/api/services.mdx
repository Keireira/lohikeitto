Returns full service details by UUID.

## Path parameters

| Parameter | Type | Description |
| --- | --- | --- |
| `service_id` | UUID | Service identifier |

    "name": "Adguard",

    },
    "category_id": "44444444-0000-0000-0000-000000000011",
    "category": "VPN & Security",
    "logo_url": "https://cdn.example.com/adguard.webp",
    "links": {
      "website": "https://adguard.com"

## Notes

- `category_id` is omitted when not set
- `ref_link` is omitted when `null`
- Returns `404` if service not found
