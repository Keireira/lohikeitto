Lohikeitto (Soup) is an HTTP API for managing a service catalog. It provides fuzzy search with Brandfetch fallback, service details with logos and brand colors, and an admin panel for managing services and categories.

## Base URL

```
https://soup.uha.app
```

## Key features

- **Fuzzy search** — trigram-based search with automatic Brandfetch fallback
- **Service catalog** — verified services with logos, brand colors, links, and categories
- **Logo management** — automatic logo sync from Brandfetch and logo.dev to Cloudflare R2
- **Country filtering** — preload services available in a given country
- **Admin panel** — Next.js app for managing services, logos, categories, and SQL export

## Response format

All responses use a unified JSON envelope:

```json
// Success
{ "status": "success", "data": T }

// Error
{ "status": "error", "code": N, "message": "..." }
```

## Tech stack

- **Backend**: Rust (Axum, SQLx, Tokio)
- **Admin panel**: Next.js 16 (React 19, TypeScript, Tailwind CSS)
- **Database**: PostgreSQL
- **Storage**: Cloudflare R2 (S3-compatible)
- **External APIs**: Brandfetch, logo.dev
