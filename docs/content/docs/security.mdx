## Authentication

Admin endpoints are protected by a Bearer token with constant-time comparison. Set `ADMIN_TOKEN` in your environment variables.

```
Authorization: Bearer <ADMIN_TOKEN>
```

## Security headers

All responses include the following headers:

- `X-Content-Type-Options: nosniff`
- `X-Frame-Options: DENY`
- `Strict-Transport-Security: max-age=...`
- `Referrer-Policy: strict-origin-when-cross-origin`

## Limits

| Limit | Value |
| --- | --- |
| Request timeout | 30s |
| Body size limit | 2MB |
| Search query length | 200 chars max |
| Country code length | 10 chars max |
| Search results count | 10 max |

## CORS

Configurable via `CORS_ORIGIN` environment variable. Defaults to `*` — set to your domain in production.

## Observability

- Structured JSON logging via `tracing`
- Request IDs (`X-Request-Id`) for distributed tracing
